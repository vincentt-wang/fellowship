# The Fellowship

A 4-week founder fellowship inside the StarWorks community. Invisible-college model:
we back builders before the capital markets do. Batch 1 starts Aug 15, with Demo Day
at the BobaTech Summit.

Static bilingual (EN/ZH) single-page site. Live at GitHub Pages.

- `index.html` — the site (self-contained; bilingual engine, live countdown, apply modal)
- `assets/img/` — StarWorks founder-community photos

Apply submissions go to FormSubmit. Ecosystem: StarWorks × the founder community × Vincent.
